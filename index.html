Panduan Penggunaan CXONEXAM

Panduan langkah-demi-langkah bagi guru dan murid untuk menyelenggarakan ujian steril.

Panduan Guru (Teacher)
  • 1
    Pilih Menu Guru

    Buka aplikasi CXONEXAM dan pilih halaman **TEACHER/GURU** di menu awal.

  • 2
    Input Detail Ujian

    Masukkan tautan soal ujian (Google Form, CBT, dll) beserta password keluar (*exit password*).

  • 3
    Integrasi Bot Telegram

    Masukkan token API Bot Telegram dan Chat ID Anda agar pesan pelanggaran siswa langsung dikirim ke HP Anda.

  • 4
    Sebarkan QR Code

    Generate QR Code ujian terenkripsi berlapis, lalu tampilkan di depan kelas menggunakan proyektor.

Panduan Siswa (Student)
  • 1
    Isi Identitas Diri

    Jalankan aplikasi, pilih **STUDENT/SISWA**, lalu masukkan nama lengkap, kelas, grup, dan nomor absen Anda.

  • 2
    Scan QR Code Ujian

    Ketuk tombol **SCAN QR** dan arahkan kamera ke barcode ujian terenkripsi yang ditampilkan guru di layar proyektor.

  • 3
    Kunci Launcher Default

    Pilih "CXONEXAM" sebagai *Home Launcher default* ketika sistem Android memintanya untuk mengunci tombol navigasi.

  • 4
    Selesai & Input Password Exit

    Setelah selesai mengerjakan soal, panggil guru pengawas untuk memasukkan *Exit Password* agar HP Anda kembali normal.

Overview Teknis & Keamanan

Dokumentasi fungsionalitas berkas kode inti pada aplikasi Android CXONEXAM.

Komponen Kode Fungsi Spesifik Teknis Pengamanan
KioskManager.kt Screen Pinning & Lock Task Mencegah penekanan tombol Home/Back/Recent menggunakan API Lock Task internal Android.
ExamKeyboardService.kt Custom Steril IME Service Mengganti keyboard bawaan sistem dengan IME buatan sendiri tanpa clipboard, auto-correct, & voice typing.
SecurityWatchdogService.kt Overlay Detection & Sound Lock Membatalkan sound/volume change, mendeteksi background activity melayang, dan memicu notifikasi pelanggaran.
TelegramHelper.kt Telegram Integration API Mengirim alert kecurangan (nama, kelas, absen, pelanggaran) via API request bot Telegram terenkripsi HTTPS.
QRUtils.kt Enkripsi Barcode Berlapis Mengamankan tautan ujian via kombinasi algoritma **AES-128**, **GZIP**, **XOR**, dan **Reversed Base64**.
ExamDeviceAdminReceiver.kt Device Owner Administration Menjadikan CXONEXAM sebagai Administrator Utama HP agar siswa tidak bisa mematikan paksa aplikasi secara manual.
